# Dedalus Marketplace Info

## Short Description
Domain intelligence via DNS, RDAP, and certificate transparency

## Long Description
Query DNS records (A, AAAA, MX, TXT, etc.), lookup RDAP registration data for domains, and search certificate transparency logs for issued certificates. Tools: dns_query, rdap_lookup, ct_search.

## Use Cases
- Search
- DevOps
